# homebrew-understatus

Homebrew tap for [understatus](https://github.com/ictechgy/understatus) — a calm, unobtrusive macOS statusline addon for Claude Code.

## Install

```bash
brew install ictechgy/understatus/understatus
```

Then enable it in Claude Code:

```bash
understatus install
```

See the [main repository](https://github.com/ictechgy/understatus) for configuration and details.
